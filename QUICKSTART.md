## âš™ï¸ Settings Explained

### Prize Configuration

- **Quantity**: How many of this prize available
- **Color**: Wheel sector color
- **Message**: Text shown when won

### High-Value Condition

- **Enable**: Turn on/off 500K restrictions
- **Min Spins**: Must spin this many times first
- **Max Spins**: Reset counter after this many

- **Warning**: Cannot be undone!
- Resets all data to defaults
- Clears spin history
- Use before starting new event
